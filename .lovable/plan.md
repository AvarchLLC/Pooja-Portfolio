
# Portfolio Website for Pooja Ranjan

A sleek, dark-themed single-page portfolio website cloning the design from protocol-herder.replit.app.

## Sections

### 1. Navigation Bar
- Fixed top navbar with "Pooja Ranjan" branding on the left
- Navigation links: About, Work, Projects, Contact on the right
- Smooth scroll to each section

### 2. Hero Section
- Dark background with profile photo on the left
- Large bold name "Pooja Ranjan" on the right
- Subtitle: "President at ECH Institute Inc. — Protocol Coordinator — Blockchain Educator"
- "Get in Touch" button + social icons (X/Twitter, LinkedIn, GitHub, Farcaster)

### 3. About Section
- Split layout: "Building Ethereum since 2016" heading on the left
- Bio paragraphs on the right describing her journey since 2016, founding Avarch LLC, EtherWorld, and EIP contributions
- Stats row: 9+ Years in Ethereum, 6+ Major Projects, 8+ Network Upgrades, 100+ EIP Contributions

### 4. Work & Contributions Section
- "Shaping the Ethereum Ecosystem" heading
- Tabbed interface with 3 tabs: Media & Talks, Educational, Protocol Coordination
- Horizontally scrolling grid of YouTube video thumbnails (8 videos) that link out to YouTube

### 5. Projects & Initiatives Section
- Grid of 6 project cards, each with logo, title, and description
- Projects: Avarch LLC, EtherWorld, ECH Institute, BlockAction Explorer, EIPsInsight, Women in Ethereum Protocol
- Each card links to the respective project URL

### 6. Contact Section
- "Let's Connect" heading with description
- Two email addresses displayed
- Social media icon links (X, LinkedIn, GitHub, Farcaster)

## Design
- Dark theme throughout (dark background, white text)
- Clean, minimal aesthetic with generous spacing
- Subtle hover effects on cards and links
- Fully responsive for mobile and desktop
